We're pleased to announce the release of

- [seL4 15.0.0](https://docs.sel4.systems/releases/sel4/15.0.0): The seL4
  microkernel
- [Microkit 2.2.0](https://docs.sel4.systems/releases/microkit/2.2.0): The seL4
  Microkit for building static-architecture systems
- [CAmkES 3.12.0](https://docs.sel4.systems/releases/camkes/camkes-3.12.0):
  Component Architecture for microkernel-based Embedded Systems
- [capDL 0.5.0](https://docs.sel4.systems/releases/capdl/0.5.0): Tools for
  generating, parsing and loading capability distribution specifications
- [rust-sel4 4.0.0](https://github.com/seL4/rust-sel4/releases/tag/v4.0.0): Rust
  support for seL4 userspace. See the Rust section on the
  [seL4 docsite](https://docs.sel4.systems/projects/rust/) for more
  information.

seL4 15.0.0 and the corresponding Rust support come with new features and will
require updates if you are using the seL4 API directly. CAmkES and the capDL
tool add domain schedules as a new feature and applications will need updates if
they use the seL4 domain scheduler. The Microkit release should not require any
update of user code compared to Microkit 2.1.0.

Enjoy!
