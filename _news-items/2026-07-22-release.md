We're pleased to announce the release of

- [seL4 16.0.0](https://docs.sel4.systems/releases/sel4/16.0.0): The seL4
  microkernel
- [Microkit 2.3.0](https://docs.sel4.systems/releases/microkit/2.3.0): The seL4
  Microkit for building static-architecture systems
- [CAmkES 3.13.0](https://docs.sel4.systems/releases/camkes/camkes-3.13.0):
  Component Architecture for microkernel-based Embedded Systems
- [capDL 0.6.0](https://docs.sel4.systems/releases/capdl/0.6.0): Tools for
  generating, parsing and loading capability distribution specifications
- [rust-sel4 5.0.0](https://github.com/seL4/rust-sel4/releases/tag/v5.0.0): Rust
  support for seL4 userspace. See the Rust section on the
  [seL4 docsite](https://docs.sel4.systems/projects/rust/) for more
  information.

seL4 16.0.0 and the corresponding Rust support have breaking changes and will
require updates if you are using the seL4 API directly. CAmkES and the capDL
tool add units for domain schedules as a new feature, but are backwards
compatible. The Microkit release comes with new features and has breaking changes
for x86_64 systems.

Enjoy!
