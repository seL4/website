We're pleased to announce the release of

- [seL4 14.0.0](https://docs.sel4.systems/releases/sel4/14.0.0): The seL4
  microkernel
- [Microkit 2.1.0](https://docs.sel4.systems/releases/microkit/2.1.0): The seL4
  Microkit for building static-architecture systems
- [CAmkES 3.11.1](https://docs.sel4.systems/releases/camkes/camkes-3.11.1):
  Component Architecture for microkernel-based Embedded Systems
- [capDL 0.4.0](https://docs.sel4.systems/releases/capdl/0.4.0): Tools for
  generating, parsing and loading capability distribution specifications
- [rust-sel4 3.0.0](https://github.com/seL4/rust-sel4/releases/tag/v3.0.0): Rust
  support for seL4 userspace. See the Rust section on the
  [seL4 docsite](https://docs.sel4.systems/projects/rust/) for more
  information.

seL4 14.0.0 and the corresponding Rust support come with new features and will
require updates if you are using the seL4 API directly. Microkit and CAmkES
abstract those and should not require any update of user code compared to
Microkit 2.0.1 and CAmkES 3.11.1.

Enjoy!
